# 🪙 Collect the Coins

> 🌐 **Descarga el juego aquí:** [https://github.com/ShaimHowl/descargajuego](https://github.com/ShaimHowl/descargajuego)

¡Bienvenido/a a **Collect the Coins**!  
Un juego de plataformas desarrollado en Godot Engine donde deberás poner a prueba tus reflejos, tu precisión y tu capacidad de supervivencia. Esquiva peligros, derrota obstáculos y recoge todas las monedas antes de que el tiempo se agote.

---

## 🎮 ¿De qué va el juego?  
**Collect the Coins** es un juego en el que el objetivo principal es **recoger todas las monedas del mapa antes de que termine la cuenta atrás**.  
Pero no será tan fácil… el mapa está lleno de peligros que pondrán en riesgo tu vida en todo momento.  
Cada nivel presenta nuevos desafíos, enemigos y entornos, haciendo que cada partida sea dinámica y exigente.

---

## ⚙️ Mecánicas principales  
- 🪙 **Recolección de monedas**  
- ⏳ **Cuenta atrás**  
- ❤️ **Sistema de vida (100%)**  
- 🔄 **Progresión sin regenerar vida**  
- 🗺️ **Interfaz por mapa**

## 🛠️ Instalación y uso  
### 📋 Requisitos  
- [Godot Engine](https://godotengine.org/) (recomendado: última versión estable 4.6)  
- Git  

---

### ▶️ Pasos para probar el juego  
1. Clona el repositorio:
```bash
git clone https://github.com/ShaimHowl/STEAL-A-COINS-1.git
```
2. Abre el proyecto en Godot Engine o ejecuta el juego desde la carpeta del proyecto.  
3. Ejecuta la escena principal (`main_menu.tscn`).  
4. ¡Empieza a jugar! 🪙🚀  

---

## 🎮 Controles y menú  
- ⌨️ **ESC – Menú de pausa**  
Opciones:  
- 🔄 Reiniciar nivel  
- 🏠 Volver al menú  
- ❌ Salir del juego  

---

## 📁 Estructura del proyecto  
- `scenes/` → niveles, menú, UI  
- `scripts/` → lógica del juego  
- `assets/` → sprites, sonidos, personaje  
- `ui/` → interfaces
